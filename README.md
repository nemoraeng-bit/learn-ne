# تطبيق تعلم اللغة الهولندية - الحروف الأبجدية
## Dutch Alphabet Learning App

تطبيق تفاعلي لتعلم الحروف الأبجدية الهولندية (A-Z) مع النطق الصوتي والتمارين التفاعلية.

## المميزات ✨
- 🔤 تعلم 26 حرف هولندي
- 🔊 نطق صوتي لكل حرف
- 📝 تمارين تفاعلية
- 📊 نظام تتبع التقدم
- 🎨 واجهة مستخدم جذابة
- 📱 تطبيق متجاوب

## التكنولوجيا المستخدمة
- HTML5
- CSS3
- JavaScript (ES6)
- Web Speech API

## كيفية الاستخدام
1. افتح ملف `index.html` في المتصفح
2. اختر الحرف الذي تريد تعلمه
3. استمع للنطق الصحيح بالضغط على 🔊
4. حل التمارين التفاعلية
5. تابع تقدمك من خلال شريط التقدم

## الملفات الرئيسية
- `index.html` - الصفحة الرئيسية
- `style.css` - تنسيق وتصميم الموقع
- `app.js` - منطق التطبيق وإدارة الحالة
- `alphabet.js` - بيانات الحروف الهولندية والأمثلة

## المميزات الإضافية
- ✅ حفظ التقدم تلقائياً في التخزين المحلي
- ✅ عرض الإحصائيات (حروف تعلمتها، نسبة النجاح، عدد المحاولات)
- ✅ تصميم ديناميكي جذاب مع رسوميات ناعمة
- ✅ دعم الأجهزة المحمولة والشاشات المختلفة
- ✅ تمارين عشوائية تفاعلية

## نطق صحيح للحروف 🎵
يتم استخدام Web Speech API لنطق الحروف بطريقة احترافية:
- النطق باللغة الهولندية الطبيعية
- سرعة نطق قابلة للتحكم
